# 1. PROJECT OVERVIEW

## 1.1 Purpose (Лойиҳа мақсади)

Ушбу портал университетда тайёрланган қуйидаги илмий-ўқув адабиётларни қабул қилиш, экспертизадан ўтказиш ва нашрга тайёрлаш жараёнларини рақамлаштириш учун ишлаб чиқилади:

* Дарслик
* Ўқув қўлланма
* Монография
* Ўқув услубий қўлланма
* Услубий кўрсатма

Тизим **муаллиф, муҳаррир, эксперт ва нашриёт ходимлари ўртасида шаффоф рақамли workflow** яратади.

Портал халқаро академик платформалар логикасига яқин ишлайди.

Асосий функциялар:

* адабиётларни онлайн қабул қилиш
* ҳужжатларни текшириш
* экспертларга тақсимлаш
* илмий экспертиза
* қайта таҳрирлаш
* нашрга рухсат бериш
* архивлаш

# 3. USER ROLES AND PERMISSIONS

Portalda **4 та асосий фойдаланувчи роли** мавжуд.

---

# 3.1 Author (Муаллиф)

Муаллиф портал орқали адабиётни юклайди.

Permissions:

Create new submission
Upload required documents
Edit metadata before review
Track submission status
Submit revised version

Restrictions:

Cannot assign reviewers
Cannot approve publications

---

# 3.2 Editor (Муҳаррир)

Муҳаррир биринчи босқич текширувни амалга оширади.

Responsibilities:

Check submission completeness
Verify documentation
Assign reviewers
Request revision
Recommend decision

---

# 3.3 Reviewer / Expert (Эксперт)

Эксперт илмий баҳолашни амалга оширади.

Permissions:

Access assigned works
Download submission files
Fill expert evaluation form
Submit review decision

---

# 3.4 Admin / Publisher

Администратор тизимни бошқаради.

Permissions:

User management
Department configuration
Workflow monitoring
Reviewer assignment
Final publishing approval
